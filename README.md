# Insurance-Analytics-Dashboard

The Insurance Analytics Dashboard is an interactive Power BI solution designed to provide insights into customer demographics, policy distribution, claim performance, premium analysis, and overall business performance. The dashboard enables users to explore data dynamically through filters, drill-through functionality, and role-based access control, helping stakeholders make informed decisions efficiently.

✨ Features
🔗 Connected and imported data from SQL Server
📈 Interactive visualizations using Cards, Slicers, Bar Charts, and Line Charts
👨‍💼 Customer demographic analysis (Gender, Age Group, etc.)
💰 Premium and policy performance tracking
📊 KPI Cards for key business metrics
🎯 Dynamic filtering using slicers
🔍 Drill-through pages for detailed record analysis
🔒 Implemented Row-Level Security (RLS) for secure data access
☁️ Published reports to Power BI Service
🔄 Configured Scheduled Refresh for automated data updates
🧪 Tested and validated RLS roles in Power BI Service
📱 User-friendly and interactive dashboard design
⚡ Optimized DAX measures and report performance
